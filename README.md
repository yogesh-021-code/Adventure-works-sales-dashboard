# 📊 Adventure Works Sales Analysis Dashboard

## 🔍 Project Overview
An interactive Excel-based Sales Dashboard built using the Adventure Works dataset.  
The project follows a Star Schema data model and provides business insights through KPIs, slicers, and dynamic visualizations.

---

## 🏗 Data Model
- Fact Table: FactInternetSales
- Dimension Tables:
  - DimProduct
  - DimCustomer
  - DimDate
  - DimGeography
  - DimSalesTerritory
- 1-to-Many relationships (Star Schema)

---

## 📈 Key KPIs
- Total Revenue
- Total Profit
- Profit Margin
- Transaction Growth
- Customer & Product Profit Contribution
- Quarterly & Time Analysis
- Country-based Performance

---

## 🎯 Key Insights
- 45–50 age group contributes highest profit
- Weekdays generate major share of revenue
- United States & Australia are top contributors
- Price Above $150 category drives majority profit

---

## 🛠 Tools Used
- Microsoft Excel
- Data Modeling
- Pivot Tables
- Measures & KPI Calculations
- Interactive Slicers
